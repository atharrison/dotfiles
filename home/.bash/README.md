# .bashrc


## Installation

### 1. Clone it down

    $ git clone git@github.com:benhamill/dot_bash.git ~/.bash

### 2. Symlink it up

    $ ln -s ~/.bash/bashrc ~/.bashrc


## Local Modifications

You can have bash aliases specific to a machine, if you like:

    $ vim ~/.bash/bash_aliases.local
